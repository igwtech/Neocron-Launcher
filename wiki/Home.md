## Features

- Game installer and delta updater from CDN
- Server manager with API integration
- Proton/Wine runtime management (GE-Proton download, prefix setup)
- PAK archive format support
- DXVK, GameMode, MangoHud toggles
- Game log viewer
- Cross-platform (Linux, Windows, macOS)

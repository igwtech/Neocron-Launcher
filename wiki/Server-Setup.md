# Server Setup

## Adding Servers

1. Click **+ Add Server**
2. Enter name, description, address, and port
3. Click **Add**

![Add Server](https://raw.githubusercontent.com/igwtech/Neocron-Launcher/main/docs/screenshots/add-server.png)

## Selecting a Server

Click any server to select it. The active server is highlighted with a cyan border.

## Importing from API

1. Click **API Login** and enter credentials
2. Click **Import Servers** to fetch endpoints from the management API

![Login](https://raw.githubusercontent.com/igwtech/Neocron-Launcher/main/docs/screenshots/login.png)

## Local Server

For the Irata emulator:
1. Start: `java -jar irata-1.0-SNAPSHOT.jar`
2. Add server: address `127.0.0.1`, port `7000`
3. Select and **LAUNCH**
